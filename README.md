# Café de Fuego

**URL:** [https://majestic-toffee-2b773f.netlify.app/](https://majestic-toffee-2b773f.netlify.app/)

## Project Overview

Café de Fuego is an e-commerce landing page promoting a premium coffee product, **"The 'Maestro' Blend."**

## Key Features

* **Product Focus:** High-Intensity Roast with Bold Flavor, made from Brazilian Beans.
* **Marketing Strategy:** Focuses on a limited-time offer and premium pricing.
* **Key Information:** Highlights Aroma, Body/Mouthfeel, Flavor Profile, and Acidity.
* **Sales Focus:** Clear calls-to-action for "Buy Now" and "View Process."

## The Maestro Blend

* **Price:** ₹ 999 (Discounted from ₹ 1499)
* **Content:** 500g Bag
* **Roasting Schedule:** Monday–Thursday
